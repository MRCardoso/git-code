
## Topic tag
```
Are releases of the your project, pointers for a specific commit
stable versions
```
### list all tags
	git tag -l

### show details of the a tag
	git show name-tag

### create a new
	git tag name-tag
	
### create a annotation
	git tag -a -m "message"

### create in an old commit
	git tag -a hash-commit -m "message"

### remove locally
	git tag -d name-tag
