
### Topic tag
	git tag name-tag
> **parameters**
* -d name-tag
* -a create a annotation
* hash-commit after '-a' create in an old commit
* -m "message" set a message

### Show details of the a tag
	git show name-tag
	
### List all tags
	git tag -l
