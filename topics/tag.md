
### Create a tag
	git tag 0.1.0

### list the all tags
	git tag -l

### sent to remove repository
	git push origin master --tags

### remove a tag
#### locally
	git tag -d 0.1.0
#### remote
	git push origin :refs/tags/0.1.0
