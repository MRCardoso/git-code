## create
### without checkout it
	git branch "name of the your branch"
### with checkout it
	git checkout -b "name of the your branch"

## checkout in the branch
	git checkout "name of the your branch"

## remove 
### locally
	git branch -D "name of the your branch"
### remote
	git push origin :branch-name
