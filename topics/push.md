## Topic push

### sending to remote repository
    git push remote-reposotory name-branch
### remove a remote branch
    git push remote-reposotory :name-branch
### remove a remote tag
    git push remote-reposotory :refs/tags/name-tag
### sending tags
	  git push remote-reposotory name-branch --tags
